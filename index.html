How it works

Transfer in seconds — no accounts, no installs, no waiting

📤 Send Files
1
Drop or pick files
Drop one or multiple files — images, videos, ZIPs, anything. Add more any time before sharing the code.
2
Share the code or link
Copy the 6-char code or use WhatsApp, Telegram, Gmail etc. The receiver sees a file preview before anything transfers.
3
⚠️ Stay until they're done
Your browser is the server. Keep this tab open until you see "✅ All files received." Closing early cuts the receiver's download immediately.
📥 Receive Files
1
Open the link or enter code
Click the shared link (auto-fills the code) or switch to Receive and type the 6-char code. The sender must have their tab open when you connect.
2
Review & choose files
You'll see a preview of all files before any data transfers. Pick what you want, then click Download All or Download Selected.
3
Save to your device
Files stream in and collect below. Download them individually or use Save All. You'll get an alert if the sender disconnects mid-transfer.
🔒

Your files never leave your browser. Beam uses WebRTC — a direct encrypted channel between browsers. The signaling server only exchanges connection info, then steps away entirely.

⚡

P2P mode: both sides must be online at the same time. The sender's open browser tab is the source. If they close it, the transfer stops and you'll see a disconnection alert.

📤 Send Files
1
Drop your file & set expiry
Pick a file (or multiple), then choose how long the link stays valid — anywhere from 5 minutes to 24 hours.
2
Upload & share the link
Click Upload to Cloud. Once done, share the link via WhatsApp, email, QR code — or just copy it. No code to memorise.
3
✅ Close your tab
You're done. The file is already in cloud storage. The receiver downloads any time before the link expires — your tab doesn't need to stay open.
📥 Receive Files
1
Open the link
No code, no app, no account needed. Just open the link the sender shared — works in any browser on any device.
2
Download your file
Click ☁️ Download — the file streams directly from cloud storage to your device. No waiting for the sender to be online.
3
🔐 Single-use & auto-deleted
The link works once. The file is deleted from cloud storage immediately after download — or automatically when the expiry time passes.
🔐

Files are deleted from cloud storage immediately after download. The link is single-use — even if someone intercepts it after the first download, the file is already gone.
